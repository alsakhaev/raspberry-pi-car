# Raspberry Pi Car

A four-wheel drive car based on Raspberry Pi board.

## Built With

* Raspberry Pi 3B+
* Raspberry Pi Camera Module v2
* L298N Dual H-bridge Motor Driver Board
* 4WD Car Platform
